Para acceder juego “sopa de letras” se debe ejecutar el archivo “menu.py”, a partir del cual se puede elegir si entrar a la configuración, si se trata del docente, o jugar.

Se incluyen cuatro archivos de formato JSON: uno para la configuración del juego, otro para la configuracion de colores, otro donde se almacenarán las palabras ingresadas. De esta manera, al reiniciar el juego no deben  cargarse las palabras nuevamente, ni tampoco la configuración, a menos que así lo desee el docente.
Finalmente, el último contiene los datos de temperatura y humedad obtenidos de los sensores conectados a una rasberry, los cuales serviran para determinar el look and feel de la aplicacion sopa de letras.
